ðŸ§  Interview Notes / FAQs
Q1. What is the difference between .NET Framework and .NET Core?
.NET Framework: Windows-only, older
.NET Core / .NET 5+: Cross-platform, modern, open-source

Q2. Why is C# used in .NET?
C# is tightly integrated with the .NET runtime and libraries
It's simple, modern, and powerful for both frontend and backend apps

Q3. What is the role of CLR?
CLR (Common Language Runtime) executes your C# code
Handles memory management, security, type safety, exception handling
