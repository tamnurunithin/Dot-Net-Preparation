ðŸ”¹ What is an Object?
- An object is an instance of a class.
- Objects are used to access class members.

Example:
Car c = new Car();
c.brand = "Toyota";
c.speed = 120;
c.Drive();

ðŸ”¹ Access Modifiers in C#

| Modifier             | Description                                               |
|----------------------|-----------------------------------------------------------|
| public               | Accessible from anywhere                                  |
| private              | Accessible only within the same class                     |
| protected            | Accessible in the class and its derived (child) classes   |
| internal             | Accessible within the same project/assembly               |
| protected internal   | Accessible in same assembly or derived class              |
| private protected    | Accessible in derived class within the same assembly      |

Example:
class Person
{
    public string name;
    private int age;
    protected string gender;

    public void SetAge(int a)
    {
        age = a;
    }

    public void GetAge()
    {
        Console.WriteLine("Age: " + age);
    }
}

class Student
{
    public string name;
    private int rollNumber;

    public void SetRollNumber(int r)
    {
        rollNumber = r;
    }

    public void Display()
    {
        Console.WriteLine($"Name: {name}, Roll Number: {rollNumber}");
    }
}

class Program
{
    static void Main()
    {
        Student s = new Student();
        s.name = "Nithin";
        s.SetRollNumber(101);
        s.Display();
    }
}
